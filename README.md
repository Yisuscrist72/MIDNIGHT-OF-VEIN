**Midnight of Vein** es la página oficial de presentación para el videojuego homónimo, un *Hack & Slash* de plataformas que desafía los límites entre el 2D y el 3D en un entorno gótico y arcano.

---

## 👁️ Visión del Proyecto
Este sitio web ha sido diseñado bajo una estética **Y2K Cyber-Goth**, fusionando la elegancia oscura de los vampiros con la crudeza tecnológica de finales de los 90. El objetivo es proporcionar una experiencia inmersiva que refleje la dualidad dimensional del juego.

## 🛠️ Tecnologías Utilizadas
- **HTML5 Semántico:** Para una estructura sólida y accesible.
- **Tailwind CSS:** Framework principal para el diseño *responsive* y utilidades rápidas.
- **Vanilla JavaScript:** Para animaciones de scroll, efectos parallax e interacciones dinámicas.
- **CSS3 Custom:** Implementación de efectos complejos:
  - ⚡ **Glitch Effect:** Distorsión visual en títulos.
  - 💿 **Liquid Chrome:** Texto con efecto de metal líquido animado.
  - 📺 **CRT Scanlines:** Filtro de monitor retro sobre toda la interfaz.

## 🕹️ Secciones Principales
1. **Hero Depth:** Introducción con efecto parallax y llamado a la acción.
2. **Dimensional Shift:** Sección informativa sobre la mecánica 2D/3D con UI de ventana retro.
3. **Gameplay Core:** Grid interactivo detallando Combate, Plataformas y Puzzles.
4. **Blood Gallery:** Muestra visual del arte del juego con filtros dinámicos.

## 🚀 Instalación y Uso
No requiere dependencias externas pesadas. Para visualizarlo localmente:

1. Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/midnight-of-vein.git
   ```
2. Abre el archivo `index.html` en tu navegador preferido.
